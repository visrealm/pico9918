## Deploying to GitHub Pages

To make these tools available online:

1. Ensure the `docs/` folder is in your repository
2. Go to Settings â†’ Pages in your GitHub repository
3. Set Source to "Deploy from a branch"
4. Select branch `main` (or `master`) and folder `/docs`
5. Click Save
6. The tools will be available at `https://<username>.github.io/<repository>/`

For the pico9918 repository: `https://visrealm.github.io/pico9918/`
