How to Apply

The generated UF2 contains only your settings — not the firmware itself. Drag and drop it onto your PICO9918 the same way you'd flash firmware (hold BOOTSEL while connecting USB, then copy the file to the RPI-RP2 / RP2350 drive). Your existing firmware stays in place; only the configuration block in flash is updated.

If you've already flashed firmware on the device, applying a config UF2 won't downgrade or replace it. To update firmware itself, grab the latest release UF2 from GitHub.
