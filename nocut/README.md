## PICO9918 no-cut mods

No-cut mods allow you to install a PICO9918 without making any permanent modifications to the case of your retro computer. Each mod provides 3D-printable parts (and in some cases a custom PCB) to cleanly route the PICO9918's video and audio connectors through the rear panel.

### Supported systems

| System | Details |
|--------|---------|
| [TI-99/4A](ti99/) | Custom PCB and 3D-printed enclosure replacing the A/V DIN socket |
| [CreatiVision](creativision/) | 3D-printed rear panel cover with VGA/RCA cutouts |
| [NABU](nabu/) | 3D-printed rear panel cover with VGA or HDMI cutouts (with optional USB boot access) |
